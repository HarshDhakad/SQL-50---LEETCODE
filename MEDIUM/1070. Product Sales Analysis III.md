# 1070. Product Sales Analysis III

**Difficulty:** Medium

---

## ðŸ§¾ Tables

### Sales

| Column Name | Type  |
|-------------|-------|
| sale_id     | int   |
| product_id  | int   |
| year        | int   |
| quantity    | int   |
| price       | int   |

- `(sale_id, year)` is the **primary key**.
- `product_id` is a **foreign key** to the **Product** table.
- Each row shows a sale on a product in a certain year.
- `price` is per unit.

---

### Product

| Column Name  | Type    |
|--------------|---------|
| product_id   | int     |
| product_name | varchar |

- `product_id` is the **primary key**.
- Each row indicates the product name for each product.

---

## ðŸŽ¯ Task

Write a solution to **select** the `product_id`, `year` (renamed as `first_year`), `quantity`, and `price` for the **first year of every product sold**.

- If any product is sold **multiple times** in its first year, **return all those sales separately**.

Return the result in **any order**.

---

## ðŸ“¥ Example

### Input

**Sales Table:**

| sale_id | product_id | year | quantity | price |
|---------|------------|------|----------|-------|
| 1       | 100        | 2008 | 10       | 5000  |
| 2       | 100        | 2009 | 12       | 5000  |
| 7       | 200        | 2011 | 15       | 9000  |

**Product Table:**

| product_id | product_name |
|------------|--------------|
| 100        | Nokia        |
| 200        | Apple        |
| 300        | Samsung      |

---

### Output

| product_id | first_year | quantity | price |
|------------|------------|----------|-------|
| 100        | 2008        | 10       | 5000  |
| 200        | 2011        | 15       | 9000  |

---

## âœ… SQL Solution

```sql
SELECT product_id,year as first_year,quantity,price 
FROM Sales
WHERE (product_id,year) IN (
    SELECT product_id,MIN(year) FROM Sales 
    GROUP BY product_id
)
```
